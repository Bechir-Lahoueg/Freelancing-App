## Tech Stack

### Client
- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Three.js (3D animations)
- Lucide React (Icons)

### Server
- Node.js
- Express
- MongoDB
- Passport.js

## Installation

### Client Setup
```bash
cd client
npm install
npm run dev
```

### Server Setup
```bash
cd server
npm install
npm start
```
